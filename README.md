To-Do Pro Ultra - Cloud Sync

O aplicație modernă de gestionare a sarcinilor, disponibilă atât ca site web, cât și ca extensie de browser, cu sincronizare în timp real prin Firebase.

Link Live

Poți accesa aplicația aici: teodorfilea.github.io/To-do

Funcționalități principale

Sincronizare în Cloud: Adaugă un task pe telefon și acesta apare instantaneu în extensia de pe calculator.

Dark Mode: Interfață adaptabilă pentru zi sau noapte.

Priorități: Setează nivelul de urgență (Scăzut, Mediu, Ridicat) cu coduri de culori.

Filtrare: Vizualizează doar task-urile active sau pe cele terminate.

Editare prin dublu-click: Modifică textul oricărui task direct în listă.

Cum se folosește?
Adăugarea unui task:

Scrie textul în căsuța "Ce facem azi?".

(Opțional) Alege o dată și o oră limită.

Selectează prioritatea și apasă pe butonul Adaugă.

Gestionare:

Bifare: Apasă pe textul unui task pentru a-l marca ca fiind terminat (va apărea tăiat).

Editare: Dacă ai greșit ceva, dă dublu-click pe text, modifică-l și apasă în afara lui pentru a salva.

Ștergere: Apasă pe butonul Șterge din dreptul task-ului.

Filtre: Folosește butoanele de sub secțiunea de adăugare pentru a vedea doar ce te interesează.

Temă: Apasă pe iconița cu lună/soare din colțul dreapta sus pentru a schimba aspectul aplicației.
